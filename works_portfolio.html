概要
自分のポートフォリオサイト
目的
自分の経歴・スキル・デザインを目指すきっかけなどを伝える
ターゲット
採用担当者・デザイナー
心掛けたこと
これまでブライダルやカスタマーサポート、更に海外で見てきた経験の色の多さと、どんな環境であっても、たとえ言葉が通じなくても明るく振る舞い、仲間と一緒に進んでいく自分のキャラクターを色と曲線で表現。
デザインは、シンプルに1カラムとしあくまでも制作物を見ていただけるようにした。
制作範囲
ライティング / デザイン / コーディング
制作期間
3週間
使用ツール
Figma / Visual Studio Code / photoshop/ Illustrator / GitHub
使用言語
HTML / CSS / jQuery
